# Проект по основам риск-менеджмента (ФТиАД ФКН) — VaR/ES по портфелю

Проектная работа №2. Мы собрали рабочую модель рыночного риска — VaR 99% и
ES 97.5% — для портфеля из 5 ОФЗ, 10 акций и валюты (USD/EUR). Факторный
анализ, стохастические модели, Монте-Карло и честный бэктест.

> **Готовы все 6 этапов.** Данные → риск-факторы (PCA кривой, акции, валюты,
> дескриптивка) → MLE-модели (нормаль, t-Стьюдента, EWMA, GARCH(1,1)+CCC,
> AR(1)/OU) → справедливая стоимость → VaR/ES на 1 и 10 дней → rolling-бэктест
> за 2025 год. Хотите сразу к выводам? Откройте `outputs/FINAL_REPORT.md`.

## Установка и запуск

```bash
python3.11 -m venv .venv
source .venv/bin/activate
python -m pip install --upgrade pip
pip install -r requirements.txt

# 1) Проверить логику без сети:
python -m scripts.smoke_test

# 2) При необходимости заново подобрать ОФЗ (интернет к moex.ru):
python -m scripts.explore_universe
#    текущий состав уже записан в rm/config.py

# 3) Загрузить и собрать всю панель данных (кэшируется локально в data_cache/):
python -m scripts.fetch_all
#    data_cache/ не коммитится; повторный запуск читает локальный кэш

# 4) Построить риск-факторы и дескриптивку (этап 2):
python -m scripts.build_factors
#    -> outputs/factors/: PCA-нагрузки, factor_scores.csv, таблицы статистик,
#       графики (scree, нагрузки кривой, корреляции, QQ, ACF²) и SUMMARY.md

# 5) Подогнать стохастические модели и сравнить их (этап 3):
python -m scripts.build_models
#    -> outputs/models/: normal_vs_t, EWMA/GARCH-параметры, ковариации «на завтра»,
#       AR(1)/OU по ставкам, графики и SUMMARY_models.md

# 6) Оценить справедливую стоимость (этап 4):
python -m scripts.build_pricing
#    -> outputs/pricing/: цены ОФЗ vs рынок, дюрации, точность за 2025,
#       стоимость портфеля, графики и SUMMARY_pricing.md

# 7) Посчитать VaR/ES (этап 5):
python -m scripts.build_risk
#    -> outputs/risk/: risk_summary.csv, es_contributions.csv, графики,
#       SUMMARY_risk.md

# 8) Сделать бэктест VaR за 2025 (этап 6):
python -m scripts.build_backtest
#    -> outputs/backtest/: backtest_series.csv, Kupiec/Christoffersen,
#       график пробоев и SUMMARY_backtest.md

# 9) Собрать финальный markdown-отчёт:
python -m scripts.build_report
#    -> outputs/FINAL_REPORT.md

# 10) Собрать визуальный дашборд результатов:
python -m scripts.build_slides
#    -> outputs/slides/: KPI-карточки, схемы, донат структуры портфеля и
#       самодостаточный dashboard.html (все графики в одной странице)
```

Если нужно только посмотреть результат без пересчета, открывайте
`outputs/FINAL_REPORT.md` и итоговые папки `outputs/factors`, `outputs/models`,
`outputs/pricing`, `outputs/risk`, `outputs/backtest`.

## Главные результаты

* Цены ОФЗ от кривой КБД попадают близко к рынку: на 02.12.2025 средняя ошибка —
  **0.573% номинала**, по доходности — **22.4 б.п.** Для сглаженной кривой это в
  самый раз.
* Однодневный VaR 99% — до **7.38 млн руб.**, десятидневный — до **25.43 млн руб.**
  Самые осторожные числа даёт историческая симуляция.
* Бэктест 2025: **3 пробоя из 250** при ожидаемых 2.5. Kupiec доволен
  (p-value 0.758), а вот Christoffersen ловит кластеризацию пробоев
  (p-value 0.020). По Базелю — green.

## Справедливая стоимость

* **ОФЗ считаем от кривой КБД**: PV = Σ CFᵢ·DF(tᵢ). Потоки (купоны + номинал)
  берём из эмиссионного расписания MOEX, дисконт — из кривой ЦБ с интерполяцией
  по сроку.
* **Капитализация — годовая или непрерывная?** Взяли ту, что ближе к рынку
  по ошибке. Развилку фиксируем явно.
* **Точность проверяем честно**: ошибку считаем в рублях, в % номинала и в б.п.
  доходности — и не на одну дату, а на всём 2025-м. Дюрация и выпуклость — проверка
  чувствительности к ставке.
* **Акции и валюту** оцениваем линейно (цена/курс × объём). **Блэк-76** — задел
  для бонуса.

## Модели динамики

Все модели подгоняем по MLE и сравниваем по AIC/BIC — разное число параметров
учитывается само:
* **Нормаль** — это база (Башелье/GBM). **t-Стьюдента** ловит тяжёлые хвосты, ν
  оцениваем по ECME/EM. И она ожидаемо выигрывает по AIC/BIC — дескриптивка ведь
  дала df≈3.
* **EWMA** (RiskMetrics λ=0.94 и λ по MLE) и **GARCH(1,1)+CCC** отвечают за
  кластеризацию: их ковариация «на завтра» реагирует на свежие шоки.
* **AR(1)/Орнштейн–Уленбек** по ставкам — проверяем, есть ли возврат к среднему
  или это случайное блуждание. На 1–10 дней блуждания обычно хватает.
* Матрицы ковариаций «на завтра» (`*_next_cov.csv`) идут прямо в риск-движок.

## Риск-факторы

* **Ставки: приращения кривой Δy → PCA по ковариации.** Сроки в одних единицах
  (доходность), стандартизация не нужна. Три компоненты — уровень, наклон,
  кривизна (Литтерман–Шейнкман) — забирают ~96% дисперсии. Почему не хрестоматийные
  99%? Кривая ЦБ уже сглаженная, плюс шок 2022-го.
* **Акции: лог-доходности → PCA по корреляции (для описания).** А тут сжатие не
  сработало: PC1 ≈ 54%, для 90% дисперсии нужно 8 компонент из 10. Поэтому в движок
  акции идут **по отдельности** — полная ковариация всех 10 бумаг, а PC1 оставляем
  как «рыночный фактор». Сожми мы до 2 PC — занизили бы риск акций на ~35%.
* **Склейка YNDX→YDEX.** Новый тикер YDEX торгуется лишь с 24.07.2024, а нам нужен
  шок 2022-го. Поэтому склеиваем его доходности с YNDX — именно доходности, а не
  цены, чтобы разрыв уровня не попал в фактор.
* **Валюты USD/EUR: лог-доходности + PCA** (общий рубль / расхождение).
* **Дескриптивка:** Жарка–Бера и Хилл (тяжёлые хвосты), ADF+KPSS (стационарность),
  Льюнг–Бокс по квадратам (кластеризация). Все числа — в `outputs/factors/SUMMARY.md`.

## Данные

* **Котировки акций, облигаций, индексов** — MOEX ISS API (`iss.moex.com`),
  свободный доступ. Первоисточник — итоги торгов самой Московской биржи. Чистая
  цена облигации идёт в % от номинала (`CLOSE`), НКД — отдельной колонкой
  (`ACCINT`); «грязная» цена = `CLOSE%·Номинал/100 + НКД`.
* **Купоны и погашения** — MOEX ISS `bondization` (эмиссионные документы,
  агрегированные биржей).
* **Кривая ставок (КБД / G-curve)** — ЦБ РФ считает её по сделкам с ОФЗ на бирже
  (модель Свенссона). То есть в основе — те же котировки ОФЗ.
* **Курсы USD/EUR** — официальные фиксинги ЦБ. И снова та самая оговорка: с
  13.06.2024 биржевые торги валютой на MOEX остановлены, и курс ЦБ с этого момента
  считается по внебиржевым данным банков.
