Теоретический контекст курса

Место решения в методологии курса

🎯

Causal inference vs предсказание

Подбор вклада — ранжирование по детерминированным формулам, не причинный вывод; причинность нужна для оценки бизнес-эффекта (A/B). Athey & Imbens (2019).

⚖️

ML-модель vs эконометрика

ML оптимизирует предсказание / ранжирование, эконометрика — интерпретируемость; ядро ближе к прозрачной модели правил, LLM — слой UX.

🧩

ML-solution-driven

Решение принимает детерминированное ядро (управляемый модельный риск), LLM не в критическом пути → управление сложностью.

🏦

Модельный риск (MRM)

ЦБ РФ (2025), SR 11-7, KPMG, Deloitte — источники риска данных / модели / использования, валидация, мониторинг.
